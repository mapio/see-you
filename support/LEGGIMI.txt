Questo file contiene una breve guida per gli impazienti.

Istruzioni più dettagliate, di cui si raccomanda a tutti la lettura,
possono essere accedute invocando il browser digitando nel terminale il
comando

	firefox	http://santini.di.unimi.it/d/labprog/esame.html &

Gli impazienti pososno viceversa seguire queste brevi indicazioni:

	0) invocate il comando 'firma' seguito dal vostro numero di
	   matricola, ad esempio, se il vostro numero di matricola è
	   123456, date il comando

	   	firma 123456

	1) portatevi nella directory esercizio-1,

	2) leggete la traccia dell'esercizio 'Testo.txt',

	3) implementate la vostra soluzione in uno (o più) file (salvati
	   nella directory corrente, con nome qualunque purché con estensione
	   '.c',  oppure '.java'), avendo cura di definire una ed una sola
	   funzione, o metodo, di nome main. Se necessario potete accedere
	   tramite il browser alla documentazione on-line disponibile
	   all'indirizzo

		http://santini.di.unimi.it/d/labprog/esame.html

	4) eseguite il comando:

		make test

	   se l'esecuzione del comando riporta errori (di compilazione,
	   o esecuzione), correggeteli prima di procedere,

	5) una vota che la verifica ha successo, consegnate il vostro
	   materiale con il comando

		consegna

	   si raccomanda di consegnare ogni soluzione appena ottenuta
	   per evitare potenziali perdite di dati,

	6) passate a considerare l'esercizio successivo seguendo di nuovo
	   queste istruzioni dal punto 1) in poi.
